Fast path for query API and webhook integration.

This prototype focuses on the first ten minutes of a developer integration: understand the release model, fetch filtered changes, and preview delivery payloads.

1

Get the latest release

curl https://your-domain.vercel.app/api/v1/releases/latest
2

Filter updates

curl "https://your-domain.vercel.app/api/v1/updates?marketplace=SG&productType=COFFEE"
3

Create a subscription

curl -X POST https://your-domain.vercel.app/api/v1/subscriptions \
  -H "Content-Type: application/json" \
  -d '{"url":"https://example.com/webhooks/spapi","marketplaces":["US","SG"]}'
4

Preview a test delivery

curl -X POST https://your-domain.vercel.app/api/v1/subscriptions/test \
  -H "Content-Type: application/json" \
  -d '{"url":"https://example.com/webhooks/spapi"}'
